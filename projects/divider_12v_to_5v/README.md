# 电阻分压器 — 12V 转 5V @ 20mA

> 双击 `circuit.asc` 在 LTspice 中打开，点 Run（▶）即可仿真。

## 这个电路做什么

最简单的"降压"电路：两个电阻串联，在中间抽头取出较低的电压。这里实现 12V → 5V。

> 💡 **注意**：分压器适合基准电压、信号缩放等**轻负载**场景。如果驱动实际负载（如电机、LED 排），电压会被拉低——那时需要用稳压器（LDO/DC-DC）。

## 在 LTspice 中看的效果

**打开 `circuit.asc` 后你会看到：**

- 左侧是直流电压源 V1（标 "DC 12"）——提供 12V
- 中间是上臂电阻 R1 = 350Ω
- 右侧是下臂电阻 R2 = 250Ω（接地）
- 导线连接，底部有接地符号
- 下方有文字 `.op`（Operating Point 分析指令）

**点 Run 后：** LTspice 自动计算直流工作点。没有波形——这是直流分析。结果在菜单 View → SPICE Error Log（或 Ctrl+L）里看到，显示 `V(out) = 5.0V`。

也可以在波形窗口右键 → Add Traces → 选 `V(out)`，看电压值。

| 指标 | 实测值 | 目标 | 达标 |
|------|--------|------|------|
| 输出电压 | 5.0 V | 5 V | ✓ |

## 电路参数

| 元件 | 值 | 作用 |
|------|-----|------|
| V1 | 12V DC | 输入电源 |
| R1 | 350 Ω | 上臂电阻（降压 7V） |
| R2 | 250 Ω | 下臂电阻（分压 5V） |

**公式**：Vout = Vin × R2/(R1+R2)；I = Vin/(R1+R2) = 12/(350+250) = 20mA

## 如何修改

右键 R1 或 R2 改数值，重新 Run。或跟 agent 说"我要把 24V 降到 3.3V"。
