# Colpitts LC 振荡器 — 谐振频率 500 kHz

> 双击 `circuit.asc` 在 LTspice 中打开，点 Run（▶）即可看振荡波形。

## 这个电路做什么

振荡器——自己产生交流信号，不需要输入信号。原理类似"推秋千"：三极管提供能量补充，LC 谐振腔决定振荡频率。这个设计目标是 ~500 kHz。

## 在 LTspice 中看的效果

**打开 `circuit.asc` 后你会看到：**

- 顶部电源 VCC = 12V
- 一个 NPN 三极管 Q1（三引脚：C 集电极、B 基极、E 发射极）
- 电感 L1 = 10µH、两个电容 C1 = C2 = 20.26 nF
- 基极偏置电阻 R1=10kΩ、R2=2.7kΩ，还有集电极/发射极电阻
- 底部接地符号
- 下方文字：
  - `.ic V(coll)=0.1`（给电路一点"推"力帮助起振）
  - `.model QNPN NPN(BF=150 ...)`（三极管行为模型）
  - `.tran 0 0.0002 0 5e-08`（瞬态仿真 200µs）

**点 Run 后：** 仿真完成 → 右键 → Add Traces → 选 `V(coll)` → OK。看到集电极电压波形：从 12V 直流开始，逐渐出现正弦振荡（振幅增长），最终稳定在约 1.3V 峰峰值——这就是起振过程。

> 注意选 `V(coll)` 而不是 `V(out)`——这个电路的振荡信号在集电极。输出节点会列出所有可选的，找 `V(coll)`。

| 指标 | 实测值 | 目标 | 达标 |
|------|--------|------|------|
| 振荡 | ✓ 已起振 | 起振 | ✓ |
| 频率 | 506.36 kHz | 500 kHz | ✓ 偏差 1.27% |
| 稳定性 | 稳定（漂移 0.45%） | 稳定 | ✓ |

## 电路参数

| 元件 | 值 | 作用 |
|------|-----|------|
| L1 | 10 µH | 谐振电感（决定振荡频率） |
| C1, C2 | 20.26 nF | Colpitts 反馈电容（C1·C2/(C1+C2) 和 L1 共振） |
| Q1 | NPN (BF=150) | 放大器件，补充能量维持振荡 |
| R1, R2 | 10kΩ, 2.7kΩ | 基极偏置分压 |
| RC, RE | 220Ω, 470Ω | 集电极/发射极电阻 |

**公式**：f = 1/(2π·√(L·Cs))，其中 Cs = C1·C2/(C1+C2)。偏差 1.27% 来自三极管的实际负载效应——正常。

## 如何修改

改频率：右键 L1 或 C1/C2 修改数值，重新 Run。注意 C1 和 C2 改为相同值。或跟 agent 说"我要 1MHz 的振荡器"。
