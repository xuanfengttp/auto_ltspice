# RC 高通滤波器 — 截止频率 1 kHz

> 双击 `circuit.asc` 在 LTspice 中打开，点 Run（▶）即可看波形。

## 这个电路做什么

高通滤波器——只让**高于 1 kHz** 的信号通过，低于 1 kHz 的被衰减。用于去除信号中的低频干扰（如电源嗡嗡声 50Hz / 60Hz）。

## 在 LTspice 中看的效果

**打开 `circuit.asc` 后你会看到：**

- 左侧是电压源 V1（标 "AC 1"）
- 中间是电容 C1 = 15.92 nF（串联在信号路径上）
- 右侧是电阻 R1 = 10 kΩ（接地）
- 元件之间有导线连接，底部有接地符号
- 下方有文字 `.ac dec 20 10 100000`（扫频指令）

注意和低通的差别：**电容和电阻位置互换了**——电容串在信号路径上（让高频通过），电阻接地（让低频旁路）。

**点 Run 后：** 右键 → Add Traces → 选 `V(out)` → 看到频率响应：低频被抑制（增益很低），过了 ~1 kHz 后增益升到 0 dB——这就是"高通"。

| 指标 | 实测值 | 目标 | 达标 |
|------|--------|------|------|
| -3dB 截止频率 | 1002 Hz | 1000 Hz | ✓ 偏差 0.22% |
| 通带增益 | -0.0 dB | 0 dB | ✓ |

## 电路参数

| 元件 | 值 | 作用 |
|------|-----|------|
| C1 | 15.92 nF | 串联在信号路径——频率越高越容易通过 |
| R1 | 10 kΩ | 接地——低频信号通过它被旁路到地 |

**公式**：fc = 1/(2π·R·C)

## 如何修改

右键 C1 或 R1 修改数值，重新 Run。或跟 agent 说"我要 500Hz 的高通"。
