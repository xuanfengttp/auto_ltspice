# RC 低通滤波器 — 截止频率 2 kHz

> 双击 `circuit.asc` 在 LTspice 中打开，点 Run（▶）即可看波形。

## 这个电路做什么

低通滤波器——只让**低于 2 kHz** 的信号通过，高于 2 kHz 的被衰减。用于滤除音频中的高频噪声。

## 在 LTspice 中看的效果

**打开 `circuit.asc` 后你会看到：**

- 左侧是电压源 V1（标 "AC 1"），它提供 1V 的扫频信号
- 中间是电阻 R1 = 1000Ω
- 右侧是电容 C1 = 79.58 nF（接地）
- 三个元件之间有导线连接，底部有接地符号
- 下方有文字 `.ac dec 20 20 200000`（扫频指令，告诉 LTspice 从 20Hz 扫到 200kHz）

**点 Run（▶ 按钮）后：**

1. 仿真自动完成（不到 1 秒）
2. 弹出空白波形窗口 → 右键 → Add Traces → 选 `V(out)` → OK
3. 看到频率响应曲线：从 0 dB 开始，过了 ~2 kHz 开始明显下降——这就是"低通"

| 指标 | 实测值 | 目标 | 达标 |
|------|--------|------|------|
| -3dB 截止频率 | 1995 Hz | 2000 Hz | ✓ 偏差 0.23% |
| 通带增益 | -0.0 dB | 0 dB | ✓ |

## 电路参数

| 元件 | 值 | 作用 |
|------|-----|------|
| R1 | 1 kΩ | 限流，和 C1 决定截止频率 |
| C1 | 79.58 nF | 对高频呈现低阻抗（把高频旁路到地） |

**公式**：fc = 1/(2π·R·C)

## 如何修改

想在 LTspice 里改成其他截止频率？右键 R1 或 C1 修改数值，重新点 Run。

或者跟 AI agent 说："帮我设计一个截止频率 5kHz 的低通滤波器"，agent 会自动生成。
