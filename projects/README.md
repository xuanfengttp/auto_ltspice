# 示范项目 — 电路小白 5 秒上手

这里是 `ltspice-circuit-agent` skill 全自动生成的 4 个真实电路案例。

**你只需要做一件事：双击下面任意一个 `.asc` 文件，LTspice 会自动打开，显示完整电路原理图，点 Run（或按 Ctrl+R）就能看到仿真波形。**

不需要懂电路，不需要碰其他文件。

---

## 4 个电路

| 电路 | 点这里打开 | 做什么用的 |
|------|-----------|-----------|
| 低通滤波器 | [打开 `lowpass_2khz/circuit.asc`](lowpass_2khz/circuit.asc) | 只让低频信号通过，滤掉高频噪声（像筛子，小颗粒过去） |
| 高通滤波器 | [打开 `highpass_1khz/circuit.asc`](highpass_1khz/circuit.asc) | 只让高频信号通过，滤掉低频干扰（反过来） |
| 电阻分压器 | [打开 `divider_12v_to_5v/circuit.asc`](divider_12v_to_5v/circuit.asc) | 把 12V 降到 5V——最简单的"降压"电路 |
| LC 振荡器 | [打开 `colpitts_500khz/circuit.asc`](colpitts_500khz/circuit.asc) | 自己产生交流信号的电路（像节拍器，自动振荡） |

每个电路打开后你会看到：**原理图上已经有完整连线、元件参数都标好了、仿真指令也设置好了**。直接点 Run 就行。

---

## 打开后看到什么

以低通滤波器为例，在 LTspice 里打开后：

1. **原理图区域**：电压源 V1（左侧）、电阻 R1（中间）、电容 C1（右侧），之间有导线连接，底部有接地符号。元件旁标注了数值（R1=1000Ω，C1=7.958e-08F）。

2. **下方文字**：`.ac dec 20 20 200000`——这是仿真指令（告诉 LTspice 扫哪些频率）。不用改，直接用。

3. **点 Run（▶ 按钮或 Ctrl+R）**：会自动仿真，弹出一个空白波形窗口。

4. **看波形**：在空白窗口中，鼠标移到顶部菜单出现 `Pick Visible Traces`（或右键 → Add Traces），选 `V(out)`，点 OK——看到频率响应曲线（低通了高频、保留了低频，增益从 0dB 开始在高频处下降）。

每个电路的 README 里有更详细的步骤和截图说明。

## 文件夹里其他文件是什么？（不用管，给 AI 用的）

| 文件名 | 这是什么 | 你需要碰吗 |
|--------|---------|-----------|
| `circuit.asc` | **原理图**，双击这个打开 LTspice | ✅ 这就是你要用的 |
| `circuit.cir` | 网表（纯文本电路描述），AI agent 用这个生成电路 | ❌ 不用管 |
| `circuit.raw` | 仿真波形原始数据（不是照片 raw！LTspice 自己会用） | ❌ 不用管 |
| `circuit.log` | 仿真运行日志（出错时给 AI 看的） | ❌ 不用管 |
| `metrics.json` | 指标数据（AI 读这个判断电路是否达标） | ❌ 不用管 |
| `README.md` | 详细设计说明（有截图、步骤、数字指标） | 📖 想了解细节再看 |

## 想设计你自己的电路？

你有两种方式：

**A. 跟 AI 说**（推荐，最简单）：打开 Claude Code / 你用的 agent，说"帮我设计一个 xxx 电路"，它会自动用这套工具生成。

**B. 自己改参数**：复制一个案例文件夹，改 `circuit.asc` 里的元件值（右键元件 → 改 Value），然后重新 Run。
